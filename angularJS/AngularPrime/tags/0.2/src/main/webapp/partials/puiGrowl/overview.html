Description The Mac's growl widget
Supports
  • Show a skinnable message on the right top for a certain time.
Known issues
  • Div with id 'growl' must be available in DOM structure.
  • New message removes current displayed message
  • Icon images must be specified in application
			.pui-growl-image {
			    background: url("../demo/images/messages.png") no-repeat !important;
			}
Planned
  • Support for sticky messages
  • Support for defining the showed time (now 3 sec)
